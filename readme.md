# dbxcli clone

A lightweight `dbxcli` clone created just for learning purposes.

## Instructions

### Build and run the app

```bash
make
```

Then, simply run the created executable:

```bash
./a.out
```

### Clear up the build artifacts

```bash
make clean
```
